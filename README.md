# WorkHive – Multi-Tenant Project Management System

## Overview

WorkHive is a backend system I built to support project and task management for multiple organizations.
The main focus of this project was to design a **multi-tenant architecture** where each organization’s data remains isolated and secure.

---

## Features

* Multi-tenant backend with organization-level data isolation
* JWT-based authentication and role-based access (Admin, Manager, Member)
* REST APIs for managing projects and tasks
* Basic task assignment and tracking system

---

## What I worked on

* Designed the backend architecture for handling multiple tenants
* Built 25+ REST APIs for project and task management
* Implemented authentication and RBAC
* Structured database for scalability and clean data separation

---

## Tech Stack

Node.js, Express.js, MongoDB, MySQL, JWT, React, Tailwind CSS

---

## How to run

```bash
git clone https://github.com/PranayAkuthota/WorkHive.git
cd WorkHive
npm install
npm start
```
